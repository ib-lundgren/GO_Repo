Installing a clone of Vigostr on appengine


Install the Python Appengine SDK and make sure appcfg.py and dev_appserver.py is on your path and marked as executable. You will also need Python and Git.


Get the source code for the website 

git clone git://github.com/ib-lundgren/Vigostr.git

Change the app.yaml name field to match your appspot name (unless you are an admin for Vigostr, then just leave it). Upload to appspot using appcfg.py or try it out locally with dev_appserver.py 


$ dev_appserver.py Vigostr/
$ appcfg.py update Vigostr/
